# Projet BDD

This is a docker container to run microsoft sql server on a docker container

## Docker

-   Windows

Follow the instructions to download and setup docker on windows:

https://docs.docker.com/desktop/install/windows-install/

## Run Container

Use this command on your terminal while in the root of the project.

`docker compose up`

## Connect

Host: localhost
Port: 1433
User: sa
Pass: abcpassword!23
db: tempdb

You can use any sql management tool like TablePlus
